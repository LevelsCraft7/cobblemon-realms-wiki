# ğŸ”„ Updating a Server

## Manual Update

1. Download the new server pack
2. Delete:

   * `mods/`
   * `config/`
   * `defaultconfigs/`
   * `kubejs/`
3. Replace them with the new ones
4. Never delete:

   * `world/`
   * `libraries/`
   * server `.jar`
5. Restart

Always back up `/world` before updating.
