## 🌌 Safari Dimension Biomes

{% hint style="info" %}
<p align="center">
The <strong>Safari Dimension</strong> contains <strong>18 Pokémon-type-based biomes</strong>, each with its own spawn pool.
</p>
{% endhint %}

---

## 🧭 Inspirations

Spawns are inspired by:

- 🧭 Pokémon X & Y - Friend Safari (type-based encounters)
- 🌌 Pokémon Legends: Z-A - Mega Dimensions

---

## ⚙️ Spawn Balancing

Spawn tables have been adjusted in order to:

- keep rare Pokémon as true rarities
- ensure varied and interesting exploration in each biome

---

{% hint style="warning" %}
## ⚠️ Compatibility

<p align="center">
Some listed Pokémon may not be available in base Cobblemon and require an additional datapack.
</p>
{% endhint %}

---

## 🧬 Spawn Organization

Each biome contains 4 rarity tiers:

- 🟢 Common
- 🔵 Uncommon
- 🟣 Rare
- ⭐ Ultra-rare

## ⚙️ Spawn System

Spawns are defined in:
<br>
➡️ `data/cobblemon/spawn_pool_world/`

Each biome uses the condition:
<br>
➡️ `"biomes": ["cobblesafari:<type>"]`
