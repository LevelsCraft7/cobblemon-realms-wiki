# 🥚 Wild Eggs and Incubators

{% hint style="info" %}
<p align="center">
In the safari, you can sometimes find <strong>wild nests</strong>. Right-click them to obtain an egg! These unknown eggs hatch into a rare Pokémon of a specific type, with increased shiny odds. However, they cannot hatch on their own: you need an <strong>Incubator</strong>.
</p>
{% endhint %}

---

## 🔥 Egg Incubator

Once placed:
- Right-click with an egg (wild egg or Cobbreeding egg) to insert it
- Wait for the incubation to finish (progress visible on the block or via Jade/WTHIT)
- At the end of the timer, click with **any Poké Ball** to retrieve the Pokémon

{% hint style="success" %}
<p align="center">
If you break the incubator, the block is recovered and the egg (with its progress) is saved.  
<br>
Place it again to resume incubation.
</p>
{% endhint %}

![Egg Incubator Crafting Recipe](https://raw.githubusercontent.com/LevelsCraft7/cobblemon-realms-wiki/refs/heads/main/imgwiki/CobbleSafari/incubateur_oeuf.png)

## 🧬 Cobbreeding Compatibility

Cobbreeding eggs are compatible with the incubator:

- ⏱️ Hatching time reduced by **33%**
- 🎯 Choice of Poké Ball upon retrieval

{% hint style="warning" %}
<p align="center">
If Cobbreeding is removed during incubation, encrypted data is lost, but the species and remaining time are preserved.
</p>
{% endhint %}

{% hint style="info" %}
## Datapack Compatibility

<p align="center">
The incubator is also compatible with datapacks. You can use any item as an egg, define multiple possible outcomes, and adjust hatching time as well as shiny multipliers.

For more information about the data structure, check the official page: <a href="https://cobblesafari.maxigregrze.fr/en/addons/datapack-egg-incubator">Egg Incubator</a>
</p>
{% endhint %}

## ⚙️ Configuration

The incubation system is fully configurable via the `incubator_config.json` file.
<br>
It allows you to modify wild egg hatching time as well as Cobbreeding egg incubation speed.

### Main Parameters

- `defaultWildEggHatchTimeTicks` → wild egg hatching time (default: `14400`)
- `cobbreedingHatchSpeedMultiplier` → Cobbreeding incubation speed (default: `0.66`)
