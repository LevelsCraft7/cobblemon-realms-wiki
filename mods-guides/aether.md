# 🚪 How to Access the Aether

## ✅ Portal Activation

1. Build a frame using **Glowstone**
2. Pour water into the center
3. Walk through the portal

![Aether Portal](https://raw.githubusercontent.com/LevelsCraft7/cobblemon-realms-wiki/refs/heads/main/imgwiki/The-Aether/Portail_Aether.jpg)

{% hint style="success" %}
Unlike the Nether or The Otherside, the Aether can be accessed very early in a playthrough.
{% endhint %}

# ⚔️ Main Progression

## 🧭 General Progression Loop

1. Secure the portal
2. Obtain basic equipment
3. Explore the islands
4. Clear dungeons
5. Improve mobility and survivability
6. Fight bosses
