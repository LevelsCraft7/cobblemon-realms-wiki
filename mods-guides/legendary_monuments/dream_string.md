# 🧵 Dream String

{% hint style="info" %}
<p align="center">
The <strong>Dream String</strong> is a material used to craft the <strong>Dream Catcher</strong> and can replace regular string in crafting recipes.
</p>
{% endhint %}

---

## 🧭 Main Function

The Dream String can be used for:

- 🪶 Crafting the **Dream Catcher**
- 🔁 Replacing **Minecraft String** in crafting recipes

---

## 🌙 Acquisition

The Dream String is obtained by battling Pokémon during the night.

- 🌑 5% chance to obtain a **Dream String** after a nighttime battle

---

## ⚠️ Note

- Rare item
- Only tied to nighttime battles
- Random drop
