# 🔨 Titan Hammer

{% hint style="info" %}
<p align="center">
The <strong>Titan Hammer</strong> is a powerful tool-weapon obtained from the <strong>Titan Core</strong>, combining both pickaxe and combat hammer functions.
</p>
{% endhint %}

---

## ⚙️ Main Function

The Titan Hammer has two uses:

### ⛏️ Mining mode
- Mines in a **3×3 area**
- Allows efficient excavation of large surfaces

### ⚔️ Combat mode
- Can be **charged**
- Deals **area-of-effect (AoE) damage**

---

## 🧭 Acquisition

The Titan Hammer is crafted from:

- 💠 **Titan Core**

---

## ⚠️ Progression Choice

The **Titan Core** can also be used to craft:

- 🛡️ The **Titan Pauldron**

👉 However, only one Titan Core is obtained per Regigigas statue appearance  
➡️ You must choose between:
- 🔨 Titan Hammer
- 🛡️ Titan Pauldron

---

## 📌 Note

- Hybrid item (tool + weapon)
- Strongly tied to Regigigas progression
- Versatile for exploration and combat
