# 🏺 Elemental Urns

{% hint style="info" %}
<p align="center">
<strong>The Urns</strong> allow you to summon the <strong>Legendary Birds</strong> as well as their Galarian forms.
</p>
{% endhint %}

---

## ⚙️ Main Function

Urns are used to summon the Legendary Birds.

👉 Simply meet the urn’s conditions, then activate it to trigger the summoning.

---

## 🧭 Usage Principle

Each urn requires:

- ⚔️ A set number of defeated Pokémon matching its type
- 🧪 Activation once the condition is fulfilled

---

## 🏺 Types of Urns

- 🔥 Embers Urn → Moltres / Galarian form
- ❄️ Frost Urn → Articuno / Galarian form
- ⚡ Storm Urn → Zapdos / Galarian form

---

## 🛠️ Crafting

### 🧱 Standard Urns

All urns share the same recipe:

- 🔷 A common base
- 💠 A **specific elemental stone placed in the center**

---

### 🌳 Galarian Urn

To craft a Galarian Urn:

- 🏺 1 standard Urn
- 🍎 1 **Dyna Fruit**
  - Found on the **Dyna Tree**
  - Locatable via the Arc Phone

---

## 📌 Activation Requirement

- ⚔️ Defeat the required number of Pokémon of the type specified in the urn description
- ✨ Then use the urn to summon the corresponding Pokémon
