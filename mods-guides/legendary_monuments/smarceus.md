# 📱 Arc Phone

{% hint style="info" %}
<p align="center">
The <strong>Arc Phone</strong> is a portable device that allows you to locate <strong>Legendary Structures</strong> across dimensions.
</p>
{% endhint %}

---

## ⚙️ Arc Phone Interface

The **Arc Phone** centralizes several applications in a single device.

👉 Right-click while holding the item 
<br>
👉 Navigate between applications via the on-screen buttons

---

## 🧭 Legendary Tracker

Allows you to **locate structures available in your current dimension**. 
<br>
A dropdown list displays all detectable structures.

---

## 🔒 Structure Access

Some structures require a **specific item** to be tracked.

These items may come from:
- Minecraft Vanilla
- Mega Showdown
- Legendary Monuments

---

### ❌ Missing Item
🟨 Name: gray
⛔ Status: inactive
📌 Info: required item temporarily displayed

---

### ✔️ Available Item
⬜ Name: white
🟢 Status: active
🚀 Action: immediate tracking possible

---

## 📍 Search Result

When you select a structure, the Arc Phone analyzes your current dimension.

If it detects a match, it displays the following information:

|   📌 Info  |                📜 Detail              |
| :---:       | :---:                                  |
| Coordinates |     Exact position in the dimension    |
|   Distance  |        Distance from the player        |
|  Direction  | General direction toward the structure |
