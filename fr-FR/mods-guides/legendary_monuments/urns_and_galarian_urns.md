# 🏺 Urnes Élémentaires

{% hint style="info" %}
<p align="center">
Les <strong>Urnes Élémentaires</strong> permettent d’invoquer les <strong>Oiseaux Légendaires</strong> ainsi que leurs formes de Galar.
</p>
{% endhint %}

---

## ⚙️ Fonction principale

Les Urnes servent à invoquer les Oiseaux Légendaires.

👉 Il suffit de remplir les conditions de l’urne puis de l’activer pour déclencher l’invocation.

---

## 🧭 Principe d’utilisation

Chaque urne demande :

- ⚔️ Un nombre défini de Pokémon vaincus correspondant à son type
- 🧪 Une activation une fois la condition remplie

---

## 🏺 Types d’Urnes

- 🔥 Urne des Braises → Artikodin de Galar / Sulfura selon variante
- ❄️ Urne du Givre → Artikodin / forme de Galar
- ⚡ Urne des Tempêtes → Électhor / forme de Galar

---

## 🛠️ Fabrication

### 🧱 Urnes normales

Toutes les urnes partagent la même recette :

- 🔷 Une base commune
- 💠 Une **pierre élémentaire spécifique placée au centre**

---

### 🌳 Urne Galarienne

Pour fabriquer une Urne Galarienne :

- 🏺 1 Urne normale
- 🍎 1 **Fruit Dynamax**
  - Trouvé sur l'**Arbre Dyna**
  - Localisable via le Smarceus

---

## 📌 Condition d’activation

- ⚔️ Vaincre un nombre requis de Pokémon du type indiqué dans la description de l’urne
- ✨ Puis utiliser l’urne pour invoquer le Pokémon correspondant
