## 💜 Comment nous soutenir ?

Vous pouvez aider le projet de différentes manières :

| ✨ Soutien | 📋 Description |
| :---: | :---: |
| 💜 Discord Nitro | Booster le serveur <a href="https://discord.gg/kb8NSTF45n">Discord</a> pour débloquer davantage d'avantages communautaires |
| ❤️ Patreon | Soutenir le développement et la maintenance du modpack sur <a href="https://www.patreon.com/cw/OurStoryCollection">Patreon</a> |
| 📣 Partage | Faire découvrir Cobblemon Realms à d'autres joueurs |
| 🐛 Signalements | Remonter les erreurs ou informations obsolètes du wiki |
| 💡 Suggestions | Proposer des idées d'amélioration pour le serveur, le modpack ou la documentation |
