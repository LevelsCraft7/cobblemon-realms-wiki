# 🌍 Prédispositions régionales

{% hint style="info" %}
<p align="center">
Cette catégorie regroupe les Pokémon possédant une prédisposition à évoluer vers une forme régionale spécifique. Ces variantes permettent d'accéder à certaines évolutions régionales même lorsqu'aucune pré-évolution régionale officielle n'existe.
</p>
{% endhint %}

Les Pokémon concernés :

- 🌍 Sont liés à une forme régionale particulière
- 🔄 Peuvent évoluer vers leur forme régionale associée
- ⚙️ Peuvent parfois évoluer vers leur forme classique sous certaines conditions
