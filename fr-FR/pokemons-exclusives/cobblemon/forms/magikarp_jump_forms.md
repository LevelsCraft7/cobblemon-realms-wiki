# 🎣 Formes de Pokémon : Magikarp Jump

{% hint style="info" %}
<p align="center">
Cette catégorie regroupe les motifs inspirés du jeu mobile Pokémon : Magikarp Jump. Chaque variante reprend l'un des nombreux motifs exclusifs qui étaient disponibles dans le jeu original.
</p>
{% endhint %}

Ces apparences possèdent plusieurs particularités :

- 🎣 S'obtiennent via la pêche avec une Poké Canne
- 🌍 Peuvent être trouvées dans tous les biomes
- 📈 Certains motifs sont plus fréquents dans des biomes spécifiques
- ✨ Chaque motif possède une apparence unique

---

## 📋 Motifs disponibles

Vous trouverez ci-dessous la liste des motifs de Magicarpe actuellement disponibles.

> *Cette liste pourra être complétée au fil des futures mises à jour.*
