# 🥋 Évolutions de Wushours

{% hint style="warning" %}
<p align="center">Les formes n'existent pas dans le mod officiel de Cobblemon. Il ne s'agit ni d'une erreur ni d'un oubli. Bien qu'elles puissent apparaître dans le menu <code>Équipe</code>, elles conservent leur apparence normale en jeu. Ceci est <strong>normal</strong>. Aucune image n'est donc affichée, mais les textes alternatifs sont conservés en prévision d'un éventuel ajout futur.</p>
{% endhint %}

| **Archéologie**      | Également obtenable dans le **sable suspect** à l'aide d'un **pinceau**                                                                                                                                                                                                                  |

***

### 🔄 Utilisation

* 🖱️ Clic-droit sur Wushours avec l'objet `Scroll of Darkness` pour lui attribuer le style.
* ℹ️ Le choix est définitif pour la ligne d’évolution.

> 💡 Le parchemin n'est pas consommé lors de son utilisation.

***

### 📜 Description

> Il vit dans une lointaine région montagneuse où il peaufine ses techniques et muscle ses jambes en courant le long des falaises abruptes.
