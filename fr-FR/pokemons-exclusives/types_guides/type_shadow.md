# 📖 Présentation du type Obscur

Le **type Obscur** possède ses propres mécaniques :

- ✨ Téracristallisation (Tera)
- 💠 Cristal Z exclusif
- ⚔️ Capacités dédiées
- 📊 Tableau d'efficacité spécifique
- 🌈 Capacité Dynamax signature

{% hint style="info" %}
<p align="center">
À l'heure actuelle, les seuls Pokémon de type <strong>Obscur</strong> sont <strong>Mewtwo Obscur</strong> et <strong>Lugia Obscur</strong>.
</p>
{% endhint %}

---

# 🌑 Énergie Obscure

Pour transformer un Pokémon en sa forme **Obscure**, il est nécessaire d'utiliser une **Énergie Obscure**.

- ❤️ Le Pokémon doit posséder au moins **250 points de bonheur**.
- 🔄 L'utilisation de l'objet réinitialise son bonheur à **0**.

# 🌑 Capacités du type Obscur

## 🕳️ Black Vortex
> 💠 **Capacité Z signature du type Obscur**

- Nécessite le **Shadowium-Z**.
- La puissance est déterminée par la capacité utilisée comme base.

---

## 🌈 Max Void
> ⭐ **Capacité Dynamax signature du type Obscur**

- ⚡ Augmente la **Vitesse** du lanceur et de ses alliés.
- 💥 Les dégâts sont calculés à partir de la capacité utilisée comme base.

---
