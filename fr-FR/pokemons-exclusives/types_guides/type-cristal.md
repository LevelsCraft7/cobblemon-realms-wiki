# 📖 Présentation du type Cristal

Le **type Cristal** possède ses propres mécaniques :

- ✨ Téracristallisation (Tera)
- 💠 Cristal Z exclusif
- ⚔️ Capacités dédiées
- 📊 Tableau d'efficacité spécifique
- 🌈 Capacité Dynamax signature

{% hint style="info" %}
<p align="center">
À l'heure actuelle, les seuls Pokémon de type <strong>Cristal</strong> appartiennent à la lignée évolutive d'<strong>Onix Cristal</strong>.
</p>
{% endhint %}
