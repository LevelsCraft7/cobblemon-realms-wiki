# 🌑 Necrozma — Fusions

Necrozma peut fusionner avec Solgaleo ou Lunala via des objets spécifiques, débloquant différentes formes ainsi que l’Ultra Burst.

---

## 🌞 Necrozma Crinière du Couchant (Dusk Mane)

| 📌 Informations | 📋 Détails |
|---|---|
| ⚔️ Type | Psy / Acier |
| 🔁 Fusion | Solgaleo |
| 🧬 Objet | N-Solarizer |
| 🎯 Rôle | Physique offensif |

### 🧪 Obtention

- 🔧 Obtenir le **N-Solarizer**
- ⚡ Clic droit sur **Solgaleo** pour charger l’objet
- 🎯 Utiliser sur **Necrozma** pour fusionner

## 💥 Ultra Necrozma

| 📌 Informations | 📋 Détails |
|---|---|
| ⚔️ Type | Psy / Dragon |
| 🧬 Objet | Ultranecrozium Z |
| 🎯 Condition | Forme fusion |
| 🎯 Rôle | Sweeper offensif |

### 🧪 Activation

- ⚡ Porter un **Z-Ring**
- 🔥 Équiper **Ultranecrozium Z**
- 🎮 En combat : Ultra Burst
- 🎮 Hors combat : touche **U**

## 🧾 Résumé

- 🌞 Dusk Mane → Solgaleo (offensif physique)
- 🌙 Dawn Wings → Lunala (offensif spécial)
- 💥 Ultra Necrozma → forme ultime ultra offensive
