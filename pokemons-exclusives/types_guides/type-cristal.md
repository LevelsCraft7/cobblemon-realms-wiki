# ✨💎 About the Crystal Type

## 📖 Crystal Type Overview

The **Crystal type** has its own unique mechanics:

- ✨ Exclusive Terastalization (Tera)
- 💠 Exclusive Z-Crystal
- ⚔️ Unique moves
- 📊 Custom type chart
- 🌈 Signature Max Move

{% hint style="info" %}
<p align="center">
At the moment, the only <strong>Crystal-type</strong> Pokémon belong to the <strong>Crystal Onix</strong> evolutionary line.
</p>
{% endhint %}
