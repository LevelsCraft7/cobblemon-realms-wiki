# 🎣 Pokémon: Magikarp Jump Forms

{% hint style="info" %}
<p align="center">
This category contains patterns inspired by the mobile game Pokémon: Magikarp Jump. Each variant recreates one of the many exclusive patterns that were available in the original game.
</p>
{% endhint %}

These appearances have several unique characteristics:

- 🎣 Obtained through fishing with a Poké Rod
- 🌍 Can be found in every biome
- 📈 Some patterns are more common in specific biomes
- ✨ Each pattern has a unique appearance

---

## 📋 Available Forms

Below is the list of currently available miscellaneous forms.

> *This list may be expanded in future updates.*
