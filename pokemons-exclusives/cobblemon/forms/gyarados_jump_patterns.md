# 🐉 Gyarados Jump Patterns

{% hint style="info" %}
<p align="center">
This category contains exclusive Gyarados appearances obtained from Magikarp inspired by Pokémon: Magikarp Jump. These variants were added specifically by Cobblemon.
</p>
{% endhint %}

These appearances:

- 🐉 Are obtained by evolving a Magikarp Jump variant
- 🎨 Preserve the visual identity of the original pattern
- ✨ Are exclusive to Cobblemon
- 🎣 Cannot be obtained directly through fishing
