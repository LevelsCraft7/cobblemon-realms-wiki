# 🐩 Furfrou Trims

Furfrou can change trims for a stylish appearance.

## 🎨 How to Change
- Hold required dye + right-click Furfrou with Shears.
- Dye is consumed.

## Available Trims
| Trim | Dye |
|------|-----|
| Debutante | Yellow |
| Pharaoh | Blue |
| Kabuki | Red |
| Star | Cyan |
| Matron | Purple |
| Dandy | Lime |
| Diamond | Orange |
| Heart | Pink |
| La Reine | Light Blue |
| Natural | White (default) |
